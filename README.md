# Multi-label Detoxicification


## Directories
- `ToxicData/`: Datasets, including `Jigsaw` and `RealtoxicPrompt`
- `pickles/`: Directories to on hold objects
- `extract_embeddings.py`: entrance to extract embeddings from specified model
- `test_single.py`: entrance to test perturbing effects
- `evaluate.py`: entrance to test perturbing performance on RTP dataset (custom proxy needed)
